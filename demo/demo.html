Dynamics 365 CE UI Library v1.0.0

TypeScript UI utility library for Dynamics 365 Customer Engagement from err403

Toast Notifications Ready

Toast notifications with auto-dismiss, stacking, and optional sound.

// Object-based API
err403.Toast.success({ title: "Account Saved", message: "The account has been updated successfully.", duration: 5000, sound: true });

// String-based API
err403.Toast.error("Error Title", "Error message", 4000, true);

Logger Utility Ready

Consistent console logging with styled prefixes (BUG, WAR, ERR).

console.debug(...err403.BUG, "Debug message", { data: 123 }); console.warn(...err403.WAR, "Warning message"); console.error(...err403.ERR, "Error message", errorObj);
Note: Open the browser console (F12) to see styled log outputs.

Lookup Component ‚úì

Advanced entity record lookup with table display, search, pagination, and metadata-driven columns.

Example:
err403.Lookup.open({ entity: 'account', columns: ['name', 'accountnumber', 'telephone1'], onSelect: (records) => { console.log(records); } });
Note: Lookup uses mock data in the demo. In D365, it will fetch real entity records using Xrm.WebApi.
